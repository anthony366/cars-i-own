Standard spacing values

These margin and padding values should be used for a consistent system of spacing both vertically and horizontally across patterns and pages.
Stylus/Sass variable Value Recommended use
$small .8rem (8px) Space/margin between elements referencing each other
$medium 1.6rem (16px) Space/margin between related or similar elements (form fields, paragraphs, paragraphs and headings)
$large 2.4rem (24px) Space/margin between modules or sections
$xlarge 3.2rem (32px) Space/padding around banded sections (footer, focus content)
